# 远程遥控车 - 控制通道协议

TCP 端口：**5103**，APP 主动连接服务器，连接建立后双向持续收发，无需握手。

## 通用帧格式

| 字段 | 偏移 | 长度 | 说明 |
|------|------|------|------|
| Magic | 0 | 2 | 固定 `0x5A 0xA5`，帧同步标识 |
| Type | 2 | 1 | `0x01` = 遥控数据，`0x02` = 遥测数据，`0x03` = GPS数据，`0x04` = 配置数据 |
| Length | 3 | 1 | 整帧长度（含校验和） |
| Payload | 4 | 可变 | 业务数据 |
| Checksum | Length-1 | 1 | 前 Length-1 字节的异或值 |

## 遥控数据帧（APP → 服务器，每 20ms 一帧）

```c
#pragma pack(push, 1)
typedef struct {
    uint8_t  magic[2];     // 0x5A, 0xA5
    uint8_t  type;         // 0x01
    uint8_t  length;       // 10
    uint16_t throttle;     // 油门（小端序，0~1024）
                          //   0 = 全速后退，512 = 停止，1024 = 全速前进
    uint16_t steering;     // 转向（小端序，0~1024）
                          //   0 = 全速左转，512 = 居中，1024 = 全速右转
    uint8_t  light;        // 灯光（0=关，1=低，2=中，3=高）
    uint8_t  checksum;     // 异或校验和
} ControlPacket;
#pragma pack(pop)
```

## 遥测数据帧（服务器 → APP）

```c
#pragma pack(push, 1)
typedef struct {
    uint8_t  magic[2];     // 0x5A, 0xA5
    uint8_t  type;         // 0x02
    uint8_t  length;       // 11
    int16_t  signal;       // 蜂窝信号强度 dBm（小端序，有符号）
    uint32_t voltage_mv;   // 电池电压，单位 mV（小端序，如 11800 表示 11.80V）
    uint8_t  checksum;     // 异或校验和
} TelemetryPacket;
#pragma pack(pop)
```

## GPS数据帧（服务器 → APP）

```c
#pragma pack(push, 1)
typedef struct {
    uint8_t  magic[2];     // 0x5A, 0xA5
    uint8_t  type;         // 0x03
    uint8_t  length;       // 30
    char     latitude[12]; // 纬度（ASCII 字符串，如 "39.904200"，不足补 '\0'）
    char     longitude[12];// 经度（ASCII 字符串，如 "116.40740"，不足补 '\0'）
    uint8_t  speed;        // 速度 km/h（0~255）
    uint8_t  checksum;     // 异或校验和
} GpsPacket;
#pragma pack(pop)
```

## 配置数据帧（APP → 服务器）

连接建立后发送，用于切换电调和灯光控制模式。模式切换时服务端会自动重新初始化 PWM 并重置控制状态。

```c
#pragma pack(push, 1)
typedef struct {
    uint8_t  magic[2];     // 0x5A, 0xA5
    uint8_t  type;         // 0x04
    uint8_t  length;       // 6
    uint8_t  motor_mode;   // 电机模式：0=内置电调，1=外部电调
    uint8_t  light_mode;   // 灯光模式：0=内置控制，1=外部控制
    uint8_t  checksum;     // 异或校验和
} ConfigPacket;
#pragma pack(pop)
```

### 模式说明

| motor_mode | light_mode | 说明 |
|------------|------------|------|
| 0 | 0 | 内置电调 + 内置灯光（默认） |
| 0 | 1 | 内置电调 + 外部灯光（舵机信号） |
| 1 | 0 | 外部电调（舵机信号）+ 内置灯光 |
| 1 | 1 | 外部电调 + 外部灯光（全舵机信号） |

### PWM 通道映射

| 通道 | PWM芯片 | 内置电调模式 | 外部电调模式 |
|------|---------|-------------|-------------|
| CH1 | PWM10 | 电机正转（500Hz PWM） | 电调舵机信号（50Hz） |
| CH2 | PWM9 | 转向舵机（50Hz，固定） | 转向舵机（50Hz，固定） |
| CH3 | PWM8 | 灯光（1000Hz PWM） | 灯光舵机信号（50Hz） |
| CH4 | PWM0 | 备用舵机（50Hz，固定） | 备用舵机（50Hz，固定） |
| — | PWM11 | 电机反转（500Hz PWM） | 不使用 |

> **注意**：CH2（PWM9 转向）和 CH4（PWM0 备用）始终输出标准舵机信号，不随模式变化。
