# 远程遥控车 - 控制通道协议

TCP 端口：**5102**，APP 主动连接服务器，连接建立后双向持续收发，无需握手。

## 通用帧格式

| 字段 | 偏移 | 长度 | 说明 |
|------|------|------|------|
| Magic | 0 | 2 | 固定 `0x5A 0xA5`，帧同步标识 |
| Type | 2 | 1 | `0x01` = 遥控数据，`0x02` = 遥测数据，`0x03` = GPS数据 |
| Length | 3 | 1 | 整帧长度（含校验和） |
| Payload | 4 | 可变 | 业务数据 |
| Checksum | Length-1 | 1 | 前 Length-1 字节的异或值 |

## 遥控数据帧（APP → 服务器，每 20ms 一帧）

```c
#pragma pack(push, 1)
typedef struct {
    uint8_t  magic[2];     // 0x5A, 0xA5
    uint8_t  type;         // 0x01
    uint8_t  length;       // 10
    uint16_t throttle;     // 油门（小端序，0~1024）
                          //   0 = 全速后退，512 = 停止，1024 = 全速前进
    uint16_t steering;     // 转向（小端序，0~1024）
                          //   0 = 全速左转，512 = 居中，1024 = 全速右转
    uint8_t  light;        // 灯光（0=关，1=低，2=中，3=高）
    uint8_t  checksum;     // 异或校验和
} ControlPacket;
#pragma pack(pop)
```

## 遥测数据帧（服务器 → APP）

```c
#pragma pack(push, 1)
typedef struct {
    uint8_t  magic[2];     // 0x5A, 0xA5
    uint8_t  type;         // 0x02
    uint8_t  length;       // 11
    int16_t  signal;       // 蜂窝信号强度 dBm（小端序，有符号）
    uint32_t voltage_mv;   // 电池电压，单位 mV（小端序，如 11800 表示 11.80V）
    uint8_t  checksum;     // 异或校验和
} TelemetryPacket;
#pragma pack(pop)
```

## GPS数据帧（服务器 → APP）

```c
#pragma pack(push, 1)
typedef struct {
    uint8_t  magic[2];     // 0x5A, 0xA5
    uint8_t  type;         // 0x03
    uint8_t  length;       // 25
    double   latitude;     // 纬度（小端序，IEEE 754 double，如 39.9042）
    double   longitude;    // 经度（小端序，IEEE 754 double，如 116.4074）
    float    speed;        // 速度 km/h（小端序，IEEE 754 float）
    uint8_t  checksum;     // 异或校验和
} GpsPacket;
#pragma pack(pop)
```
